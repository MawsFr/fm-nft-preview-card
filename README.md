### What I learned

I've learned that I can put more than one shadow on an element with the `box-shadow` property.

```css
.card {
  box-shadow: 0px 30px 10px 15px rgb(12, 23, 42),
    0px 40px 10px 50px rgb(13, 25, 44);
  ...
}
```
